Weather Widget App

Reasoning

The reason why I decided to create this Weather Widget App is because so that I can show how effective I am with Angular

Stack

The stack that I used to create this project is HTML5, CSS, Javascript, and Angular. The reason why I chose this stack is because I wanted to show that you can use Angular to create different applications such as a Weather app.

Lessons

I learned a few things from creating this project. One thing I learned from creating this project is how to properly fetch and display the weather information of the city from the OpenWeather Api. This project also reinforced proper composition and placement when it comes to styling the weather app.
