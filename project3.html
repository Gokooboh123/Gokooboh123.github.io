Dynamic Landing Page

Reasoning

The reason why I decided to create this Dynamic Landing Page is because so that I can display my UI skills, along with displaying proper Local Storage management

Stack

The stack that I used to create this project is HTML5, CSS and Vanilla Javascript. The reason why I chose this stack is because I wanted to show that you can create effective applications with this stack alone. Some of the challenges that I faced had to do with finding ways to allow the user to add their name and daily goal to local storage.

Lessons

I learned a few things from creating this project. One thing I learned is how to use functions and event listeners to allow the user to add their name and daily goal to local storage. I also learned how to create functions that allow the site to get the goal and the name of the user
